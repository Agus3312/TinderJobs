# Skill Registry - Tinder de Trabajo

## Compact Rules

### React & Next.js
- Use Functional Components with TypeScript interfaces.
- Use Framer Motion for animations.
- Tailwind CSS 4 for styling (no Tailwind config needed).
- Follow App Router patterns.

### UI/UX
- Premium aesthetics: gradients, glassmorphism, smooth transitions.
- Interactive elements must have hover/active states.
